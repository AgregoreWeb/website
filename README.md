# website
Website for Agregore

## Contributing

- Open a github issue if you have feature requests or suggestions for something you'd like to add
- Edit or add files in the git repo in a new branch then submit a pull request

Types of contributions that would have high impact:
- New links on the [explore page](https://github.com/AgregoreWeb/website/blob/main/explore.md), especially if you made blog posts or apps
- More [tutorials](https://github.com/AgregoreWeb/website/tree/main/docs/tutorials) for making apps based on our [process](https://github.com/AgregoreWeb/website/blob/main/docs/tutorials/process.md).
- Questions and answers added to the [frequently asked questions](https://github.com/AgregoreWeb/website/blob/main/faq.md) page.
- Fixes and improvements for wording and spelling on any pages you see.
- [Code snippets](https://github.com/AgregoreWeb/website/blob/main/docs/snippets.md) that you found useful in making apps
