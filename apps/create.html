Create or Open a Resource

or open:
What is this?

For Spellcasters:

Create a new file or open an existing one and pass it to a different web app. If you want to open an existing resource you can either paste a link into the text box, type one in and press enter, or drag a link anywhere onto the page. Otherwise you can use the Create form to make a new one.

For Spellweavers:

This is used by other apps as an open dialog so they don't have to make one themselves.. The final URL is passed to the url parameter of whatever app requested it. You can specify the viewer to use by specifying it in the url search param on this page. Like ?url=foo.html. You can specify a default name with the name parameter. You can specify a required file extension with the type parameter.
